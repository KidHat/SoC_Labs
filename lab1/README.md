# lab-riscv-asm

Вариант №8 (Николай Байдашев): найти сумму отрицательных элементов массива размером 6×6. Итоговая сумма для зашитого тестового массива равна `-75`.

## Состав
- `task8.c` — решение на C (статический 6×6 массив, вывод результата).
- `task8.S` — ручная реализация на RISC-V ASM для Venus.
- `Makefile` — сборка ELF и дампов для C-версии и, опционально, для ASM (`make asm`).
- `example.c` / `example.S` / `Makefile.txt` — демонстрационные файлы из примера.
- `ITMO_SOC_lab1.pdf` — методичка с условиями вариантов.

## Алгоритм
1. Инициализировать сумму `sum = 0`.
2. Пройти все 36 элементов массива.
3. Если элемент < 0, добавить его к `sum`.
4. Вывести итоговую сумму.

## Сборка
Требуются `riscv64-unknown-elf-gcc` и `riscv64-unknown-elf-objdump` в `PATH`.
```bash
cd github
make          # соберёт task8.elf и task8.dump из C
make asm      # дополнительно соберёт task8_asm.elf и task8_asm.dump из ASM
make clean    # очистка артефактов
```

## Запуск
1) **Через Venus (ASM):**
- Открыть https://www.kvakil.me/venus/.
- Вставить содержимое `task8.S`.
- Нажать Run. На консоль выйдет `-75` и перевод строки.

2) **Через ELF (если есть эмулятор):**
- После `make` выполнить `qemu-riscv32 task8.elf` (нужен `qemu-user` с поддержкой rv32).
- Ожидаемый вывод: `Sum of negative elements: -75`.

## Проверка
- Сумма отрицательных значений зашитого массива: `-75` (совпадает между C и ASM версиями).
- Дамп C-версии: `task8.dump`; дамп ASM-версии: `task8_asm.dump` (после `make asm`).
